html
웹 접근성을 개선하고 모든 사용자가 편리하게 이용할 수 있는 웹 페이지를 구축할 수 있습니다.
시맨틱 요소를 사용하여 구조를 명확하게 정의할 수 있습니다.
CSS
Css 3.0을 사용하여 웹 페이지의 스타일 및 레이아웃을 정의 할 수 있습니다.
다양한 웹 브라우저에서 동일하게 보이며 일관된 방식으로 작동되도록 크로스 브라우징 가이드라인을 지킵니다.
다양한 크기의 해상도에 따라 Media Query를 사용하여 최적의 경험을 제공할 수 있습니다.
Scss를 이용한 프리 프로세싱으로 코드의 가독성과 유지보수성을 향상 시킬 수 있습니다.
javascript
메뉴,슬라이드,스크롤 애니메이션 등 동적 기능을 자바스크립트와 제이쿼리로 구현 가능합니다.
Javascript 플러그인을 사용할 수 있습니다.
Ajax에 대한 이해가 있습니다.
git hub
Git hub에 작업을 업로드하여 관리할 수 있습니다.
Git hub 를 이용한 협업이 가능합니다.
Other Tools
Photoshop,Illustrator를 이용하여 창의적인 디자인을 제작할 수 있습니다.
Figma를 활용한 와이어프레임 제작할 수 있습니다.
